## Установка и настройка

### Шаг 1: Клонирование репозитория

Клонируйте проект из GitHub:

```bash
git clone https://github.com/B2ng4/Microservices.git
```

### Шаг 2: Установка docker

```bash
sudo apt install docker-ce
```


### Шаг 2:Запуск

```bash

docker-compose build up
```
